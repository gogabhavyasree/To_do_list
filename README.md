# 📝 Todo CRUD Web App

A simple and beginner-friendly Todo List web application that performs basic **CRUD (Create, Read, Update, Delete)** operations.
This project helps understand how frontend logic works with dynamic data handling.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks
* 💾 Stores tasks in browser (optional improvement: Local Storage)

---

## 🛠️ Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic

---

## 📂 Project Structure

todo-crud-app
├── index.html
├── style.css
└── script.js

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open the folder
3. Double-click on **index.html**
4. The app will open in your browser

---

## 💡 How It Works

* User enters a task in the input field
* Clicks **Add** button
* Task is displayed in the list
* User can delete the task using the delete button

---

## 📸 Output

A simple Todo interface where users can manage daily tasks easily.
